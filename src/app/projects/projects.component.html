Projetos

Alguns projetos que já realizei...

MyBank: Java, Spring e MongoDB

Aplicação SpringBoot que simula a integração e processamento de contas para Receita Federal, enviadas via arquivo .csv integrando com um banco fictício ("My Bank") utilizando Java, Spring e MongoDB.

StarBank: Microsserviços e RabbitMQ

Simula a concessão de cartão de crédito através de uma avaliação de crédito do cliente. Foi implementada uma arquitetura com service discovery, API Gateway, balanceamento de carga, comunicação síncrona com Spring Cloud e comunicação assíncrona com RabbitMQ.

Testes Unitários e Integrados

Criação de testes unitários e integrados para uma aplicação de petshop, utilizando JUnit, Mockito e Spring framework. Foi realizada a integração com a ferramenta Circle CI para a configuração de uma esteira de execução de testes do projeto.

Biblioteca: Api Rest + Api externa

Consumo de Api externas e realização de querys personalizadas com Jpa, para realização de cadastro de clientes e funcionários de uma biblioteca.
